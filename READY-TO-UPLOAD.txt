UPLOAD / UPGRADE
1. Back up your current Academy files and database.
2. Upload the contents of this ZIP over the existing Academy project.
3. Keep your existing .env file. This ZIP does not replace it.
4. Run: npm install
5. Run your normal deployment/build command.

Hostinger build command already included in package.json:
   npm run hostinger-build

Do not upload a separate node_modules folder. npm install recreates it for the server.
